Hosted remote dev boxes

BoxHaven keeps AI coding agents running after your laptop closes.

Launch Claude, Codex, Gemini, or shell work inside a managed remote tmux session. Disconnect, come back later, and reattach to the same box with the project still there.

The workflow

Create once. Reattach whenever work is ready.

BoxHaven syncs the project to a remote box, starts the agent in a persistent session, and lets you reconnect from any terminal.

bh

$ bh create api-refactor

$ bh run api-refactor codex

Codex> Working in /opt/boxhaven/project

$ bh connect api-refactor

Hosted service

A managed control plane for teams using agents every day.

Persistent sessions

Every box has a managed tmux session, so agent work survives disconnects, browser closes, and laptop sleep.

Ready images

New boxes boot from prepared snapshots with the agent runtime and development tools already installed.

Team access

Share boxes through team ownership, roles, invite links, and a browser console for visibility.

Open source core

Run the backend yourself when you need your own boundary.

The CLI, backend, VM agent, and provider integrations are open source. Self-hosted deployments keep the auth and console app without the paid-service website.
